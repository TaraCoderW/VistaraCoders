## 🆘 Troubleshooting

### Links Don't Work
**Problem**: Clicking navigation shows 404
**Solution**: 
- Check filenames are exactly: `index.html`, `Landing.html`, `AIPAGE.html`, etc.
- Verify all files are in same directory
- Clear browser cache (Ctrl+Shift+Delete)

### Page Styling Looks Broken
**Problem**: Colors/layout wrong
**Solution**:
- Hard refresh: `Ctrl+Shift+R`
- Check if Tailwind CSS loaded (check browser console)
- Verify internet connection for external assets

### Mobile Menu Not Working
**Problem**: Hamburger menu doesn't open
**Solution**:
- Check JavaScript enabled
- Open browser console for errors
- Clear cache and reload

### Something Else?
1. Open browser console: `F12`
2. Check for red error messages
3. Try different browser
4. Check README.md for help
